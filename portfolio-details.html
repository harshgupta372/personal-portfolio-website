About this music player

Project information

  • Scripted through: Vanilla JavaScript
  • Inspired by: Spotify
  • Framework used: React
  • Project URL: https://sspotify.netlify.app/

This project builds a responsive Spotify web player entirely from scratch using HTML, CSS, and JavaScript. It features a user-friendly interface with playback controls, simulated playlists, and basic visualizations. The code emphasizes clean structure and responsiveness for a seamless experience across devices. While currently focused on UI and lacking music streaming functionality, future plans include API integration and advanced features. Pull requests are welcome to contribute to this project's development !
